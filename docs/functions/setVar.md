# $setVar
Set a variable into the local storage.
## Usage
> `$setVar[name;value;table?]`
## Parameters
| Name  |     Description      |  Type  | Default value |
|-------|----------------------|--------|---------------|
| Name  | Variable name.       | String | none          |
| Value | Variable value.      | String | none          |
| Table | Local storage table. | String | none          |
